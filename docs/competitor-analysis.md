# Competitor Analysis

Generated 2026-05-04 from DuckDuckGo HTML SERP analysis across four target queries. The methodology, raw rankings, and consolidated competitor list below feed task #25 (per-competitor SEO audit) and task #26 (content-gap mapping).

## Methodology

Four target queries were run, each scoped to surface organic results competing with bizgrants.consulting:

1. `CCP advisory Singapore Career Conversion Programme consultant`
2. `Job Redesign Grant consultant Singapore`
3. `Career Conversion Programme consultant Singapore SME`
4. `Workforce Singapore grant advisory consultant`

Government domains (`*.gov.sg`, Workforce Singapore official pages, MOM, ACRA) were filtered out. Industry-federation listings (SBF, SNEF, IBF) were also excluded as they are not advisory-firm competitors.

A "consistent competitor" is defined as any firm appearing in the top 10 organic results for **two or more** of the four queries.

## BizGrants' current position

BizGrants Consulting ranks **#1 in all four target queries** in the current sample. Caveats:

- DuckDuckGo HTML SERP results are not identical to Google.sg results. Real Singapore search traffic uses Google and the rankings may differ.
- Single-IP unauthenticated bot results may be over-personalised toward simpler/exact-match domain matches; bizgrants.consulting is a strong exact-match for the target queries.
- These rankings should be reconfirmed with a real SERP API (DataForSEO, SerpApi, or similar) for higher-confidence Google.sg-localised data before declaring a definitive position.

That said, the result is consistent with the recent SEO health-score progression (59 → 79 → 88 → 91+) and the comprehensive on-page work across the last quarter.
